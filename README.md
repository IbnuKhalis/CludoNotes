# 🕵️ Cluedo Detective Notes

A simple web app for tracking Cluedo game deductions, suspects, weapons, and rooms.

Built using plain HTML, TailwindCSS, and vanilla JavaScript. No framework. Just raw determination and one gigantic `index.html`.

---

## ✨ Features

- Dynamic player management
- Detective deduction sheet
- Status cycle system:
  - `-` Empty
  - `X` Eliminated
  - `✓` Confirmed
  - `?` Suspicious
- Automatic accusation suggestion
- Dark mode support
- LocalStorage auto-save
- Responsive mobile-friendly layout
- Investigation notes section

## 🚀 How to Run

1. Download or clone this repository
2. Open `index.html` in your browser

No installation required.

Because modern web development apparently reinvented “open file in browser” into a 900MB dependency tree.
